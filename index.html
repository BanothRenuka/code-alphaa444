1. Project Overview

Learnify is an online learning platform designed to help users easily discover, enroll, and continue courses. The prototype focuses on clarity, accessibility, and simple navigation.

2. User Flow

Users land on the Home page, view their progress, explore courses, and manage their profile. Navigation buttons allow quick switching between pages, simulating a real clickable prototype.

3. UI/UX Principles Used

  • Consistent colors and typography
  • Large buttons for easy interaction
  • Minimal content per screen

4. Conclusion

This prototype demonstrates a complete mini-project including layout, interaction, and usability. It can be recreated in Figma or Adobe XD using the same structure and flow.
